## 📖 Introduction

This project focuses on analyzing retail sales data using **Microsoft Excel** and **Power BI** to uncover meaningful business insights. The project demonstrates the complete data analysis process—from data cleaning and preparation to visualization and dashboard development.

The interactive dashboards provide a clear overview of sales performance, customer purchasing behaviour, product performance, and store performance, enabling data-driven decision-making.

---

## 🎯 Project Objectives

The objectives of this project were to:

- Analyze overall retail sales performance.
- Identify the best-performing product categories.
- Compare sales across different stores.
- Examine customer purchasing behaviour by gender.
- Build an interactive dashboard for easy data exploration.
- Generate actionable business insights and recommendations.

## 📂 Dataset

The dataset contains retail sales transaction records, including:

- Order ID
- Order Date
- Product Category
- Product Name
- Sales Amount
- Quantity Sold
- Customer Gender
- Store
- Customer Information

---

## 🧹 Data Cleaning Process

The following data cleaning activities were performed:

- Removed duplicate records
- Checked for missing values
- Standardized data formats
- Corrected inconsistent entries
- Verified data accuracy
- Prepared the dataset for analysis

---

## 📈 Dashboard Features

The dashboard includes the following interactive visualizations:

- Total Sales KPI
- Total Quantity Sold
- Sales by Product Category
- Sales by Store
- Sales by Gender
- Interactive Slicers
  - Product Category
  - Store
  - Gender

Users can filter the dashboard to explore different business segments and gain deeper insights.

---

## 💡 Key Insights

The analysis revealed several important findings:

- Some product categories generated significantly higher sales than others.
- Certain stores consistently outperformed others.
- Customer purchasing patterns varied by gender.
- Interactive dashboards made it easier to monitor business performance.
- Sales trends highlighted opportunities for improving inventory management and marketing strategies.

---

## 📌 Business Recommendations

Based on the analysis, the following recommendations were made:

- Increase inventory for high-performing products.
- Strengthen marketing efforts in lower-performing stores.
- Develop targeted promotional campaigns based on customer demographics.
- Use dashboard insights to support strategic decision-making.
- Continuously monitor KPIs to improve operational performance.
